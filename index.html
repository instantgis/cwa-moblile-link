CWA map deep linking example

Click to start the CWA Android Map App and find customer 0208Y249A

The anchor tag above is formatted as:

intent://cwa.com?directLink=true&customer=0208Y249A#Intent;scheme=http;package=com.cwa.mobile;end

The scheme is http
The host is cwa.com
Query parameters follow the ? and are
directLink=true
and
customer=0208Y249A

This corresponds to the folloging Android Manifest entries
Android should look for an app witg package com.cwa.mobile responding to this intent and invoke it with no prompt.
